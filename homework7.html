Homework 7: Website Styling and Navigation

Objective: For this assignment, I focused on refining the overall design and navigation of my course website. The goal was to create a cohesive, professional-looking site with consistent styling across all pages, easy navigation, and thoughtful presentation of all coursework.

Navigation Structure: I built a consistent navigation system that appears on every page of my site. Each page includes a vertical stack of links at the bottom, styled as pill-shaped buttons with white borders. This design choice ensures visitors can easily move between assignments without getting lost. Every page also includes a prominent "Back to Home" link as the first option. For pages that are part of series (like the three Camera Raw exercises), I included direct links to related assignments to improve workflow.

Transparent Overlay

Typography Choices: I selected Gabriola for all headings because it has a hand-crafted, organic quality that reflects my background in illustration and animation. It stands out while remaining elegant and readable. For body text, I chose Segoe UI with Roboto as a fallback — these are clean, modern sans-serif fonts that ensure long paragraphs remain easy to read against the animated background. All text is white for maximum contrast against the dark overlay.

Background and Readability: The cinematic GIF background creates visual interest and mood, but moving backgrounds can interfere with text readability. I solved this by adding a semi-transparent black overlay (0.6 opacity) over the entire background, then adding an additional darker overlay (0.75 opacity) specifically for text blocks. This creates a "film strip" effect where text areas feel grounded while the background still animates around them.

Image Presentation

Image Consistency: Every image across all homework pages now has a consistent white border (4px) and subtle rounded corners. This creates visual unity across assignments and makes the gallery feel cohesive. Each image includes a caption identifying the exercise or content, helping viewers understand what they're looking at without having to read surrounding text.

Layout Structure: I used semantic HTML elements and organized content into clear sections with consistent spacing. Images are displayed in flexible grid layouts that adapt to different screen sizes using flexbox and media queries. This ensures the site is fully responsive and looks good on both desktop and mobile devices.

Site-Wide Improvements

  • External CSS: Consolidated all styling into a consistent internal CSS block on each page, ensuring visual uniformity. Future improvement will be moving to a single external stylesheet.
  • Consistent Headers: All pages use the same H1 styling with Gabriola font and white border accent.
  • Title Text Emphasis: Headers stand out with larger size, distinct font, and subtle text shadow for depth.
  • Image Sizing: All images are properly sized for web with max-width constraints to prevent layout breaking.
  • Background: Non-white background with cinematic GIF and dark overlay for enhanced appearance and readability.
  • Navigation: Every page includes consistent link structure with easy return to home page and cross-linking between related assignments.

These improvements make the site easier to navigate and more professional in appearance. The consistent design language across all pages reinforces my personal brand as an artist and designer, while the thoughtful navigation ensures that visitors can explore my coursework without frustration.
