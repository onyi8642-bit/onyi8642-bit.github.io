Homework 6: Color and Pattern

Pattern 1: Triadic Color Scheme

Color Rule: Triadic For this pattern, I used a triadic color scheme based on the Adobe Color website. The three colors are evenly spaced around the color wheel, creating a vibrant and balanced composition. I chose a combination of teal, coral, and mustard yellow. These colors have high contrast while still feeling harmonious together. The pattern itself is built from repeating geometric shapes — circles, triangles, and hexagons — arranged in a tessellating grid. I used the Pattern Options window in Illustrator to create a seamless repeat with a 4K size (4096 x 3112 px). The overlapping shapes create depth and movement, while the triadic colors ensure each element stands out without overwhelming the composition.

Pattern 2: Color Mood from Image

Color Mood: For the second pattern, I created a color mood by extracting a palette from a reference image — a photograph of a sunset over the ocean. Using the Color Mood tool in Adobe Color, I captured the deep navy blues, warm oranges, dusty rose, and soft lavender from the original photo. I then applied these colors to a pattern built from layered organic shapes inspired by ocean waves and coral formations. The pattern uses a more fluid, natural composition compared to the geometric first pattern. I created this pattern at tabloid print size (792 x 1224 pt) using Illustrator's Pattern Options with a grid offset to create a staggered, more organic repeat. The result feels both calming and dynamic, reflecting the mood of the original sunset image.

Clothing Mockup

Application: I used an online mockup generator to apply my triadic geometric pattern to a shirt. This step helps visualize how the pattern would work as a textile design. The repeating geometric shapes translate well to fabric, creating a bold, modern look. The high contrast triadic colors make the pattern pop against the garment's base, while the seamless repeat ensures the design flows naturally across seams and edges. Seeing the pattern on a 3D mockup gives a better sense of scale and how the colors interact in a real-world application.

Process Summary: Both patterns were created using Adobe Illustrator's Pattern Options tool. I started with basic geometric and organic shapes, built them into a tile, and adjusted the spacing, overlap, and offset to achieve a seamless repeat. The triadic pattern used a strict grid layout with overlapping elements to create depth. The color mood pattern used a more fluid arrangement with a brick offset to create visual interest. I saved both patterns as swatches, then applied them to artboards at the required sizes before exporting as PNGs for web.
